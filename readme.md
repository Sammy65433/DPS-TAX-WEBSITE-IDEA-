# DPS Professional Tax Services — Website

This is the website for **DPS Professional Tax Services & Realty Management**, located in Maplewood, NJ.

## 👀 View the Live Site

The easiest way to see how the site looks: open `index.html` directly in your browser, or visit the live link (add your hosting link here once deployed, e.g. Netlify/Vercel/GitHub Pages).

> If you're viewing this on GitHub and just want to see the design without downloading anything, ask for the live preview link instead of digging through the code.

## 📋 What's Included

- Tax service overview (preparation, notary, translation, immigration, money transfer, copy/fax)
- Online tax appointment booking form
- "15+ Years in Business" trust section
- Real estate partner section (RC Realty Group) with its own appointment form
- Multilingual support callouts (English, Kreyòl, Français, Español)
- Mobile-responsive layout

## 🛠️ How to Run Locally

1. Download or clone this repository.
2. Open the folder in VS Code (or any editor).
3. Open `index.html` in your browser — no install or build step needed.

If you want live-reload while editing, install the **Live Server** extension in VS Code and click "Go Live" at the bottom right.

## ✏️ Making Changes

- **Text/content:** edit directly inside `index.html`.
- **Colors/fonts/spacing:** edit `style.css`.
- **Logo:** replace `public/DPS LOGO1.png` with a new file of the same name, or update the `src` path in `index.html` if you rename it.
